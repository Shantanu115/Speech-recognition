# ASR Benchmark — Indian Conversational Speech
Vahan.ai intern assignment. Benchmarks Deepgram Nova-2, Whisper Medium, IndicWhisper, and Sarvam AI on Hindi/Hinglish locality-name recognition.

Full write-up in `Report.pdf`.

## Run

Open `asr_final.ipynb` in Google Colab. Add two secrets (🔑 sidebar):
- `DEEPGRAM_API_KEY` — [console.deepgram.com](https://console.deepgram.com)
- `SARVAM_API_KEY` — [dashboard.sarvam.ai](https://dashboard.sarvam.ai)

Audio files: [Google Drive](https://drive.google.com/drive/folders/183o9Eb7ZAeQhFOZPDDoagtqiinUqdVp8?usp=sharing) — copy the `asr/` folder into your `MyDrive/` and run top to bottom.
